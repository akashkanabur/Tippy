# 💸 Tippy – My First Android App

Tippy is a simple and beginner-friendly Android application that calculates the tip amount based on the total bill and selected tip percentage.

This is my **first ever app**, built as a starting point in my journey as an Android developer 🚀

---

## 📱 Features

- Enter bill amount  
- Choose tip percentage (e.g., 15%, 18%, 20%)  
- Instantly calculate tip amount  
- Clean and minimal UI  
- Beginner-friendly logic  

---

## 🛠️ Tech Stack

- **Language:** Java / Kotlin  
- **UI:** XML Layouts  
- **Platform:** Android Studio
